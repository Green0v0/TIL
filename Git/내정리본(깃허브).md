# 깃허브

깃은 내컴에서 쓰는 프로그램

깃허브는 원드라이브 같은 플랫폼(싸피에서는 깃랩 사용,,)

1. 리파지토리 생성(무조건 프라이빗)
2. 깃허브 연동,, til 폴더에서 깃배시 열고
3. git init
4. git remote add origin 깃헙저장소주소(시프트 인서트 복붙)
5. git remote -v  쳐봐서 연결 확인 (패치, 푸시)
6. git push origin master 로 깃헙올리기!
7. 뜨는창 예예 선택,,~~
