# 마우스 우클릭으로 GIT BASH

### .git 폴더가 잇는곳에서 수행! 



git init 깃으로 관리해달란뜻

git add. 커밋할 목록에 추가(스테이징에리어로 온것) (add . 이면 해당 경로의 `모든` 변경사항을 stagin 												area로 넣어요)

git commit -m "first commit"  커밋 만들기

git log 로 커밋이 잘되었는지 확인!

이 것을 깃배쉬에서 하는것,,, CLI라고 부름(일반적인 사용자는 GUI)

pwd 내가일하는곳의 주소를 알려주는것

ls 치면 파일명 나옴 무슨파일이잇는지

보기-숨김항목 체크하면 깃 나옴

ls -a 하면 깃숨겨진것까지( ./ 현재위치 ../이전위치 혹은 상위폴더)

cd ..  상위폴더로 가겟다는뜻

cd 원하는폴더명 ->원하는 폴더로 간다는뜻

clear 화면 깨끗하게 지워줘 혹은 컨트롤+k..?

touch 원하는 파일명 -> 파일만드는 명령어

mkdir 원하는 폴더명 -> 폴더 생성

rm 지우고싶은파일명 ->파일삭제

git status ->너의 상태가 어떠니

메세지 작성할때 modify (내용수정함),  add (신규파일추가)

git add a.txt 이케하면 a파일만 추가

commit이 작업한 순간의 시점을 기록, 사진을 찍는다고 생각하면됨



git commit을 하는 이유는 기록을 남기기 위함, 남겨놓지 않으면 이전 버전에서의 변경사항을 모르기 때문에
